Manuel d'utilisateur

Ce manuel détaille le fonctionnement du logiciel de modélisation avec des simploïdes.

L'application est composée de 4 parties : la boîte à outils, les menus, l'arbre des simploïdes et le canvas.

Boîte à outils

La partie droite de l'interface regroupe l'ensemble des fonctionnalités suivantes : ajout de simploïdes, opérations de construction, vue explosée et choix du type de plongement.

Ajout d'un simploïde

Cette sous-partie de la boîte à outils permet la création d'un nouveau simploïde.
Les simploïdes de base (à savoir point, segment, triangle, carré, tétraèdre, cube et prisme) sont proposés.

L'ajout d'un simploïde se fait comme suit :

  1. Choisir un simploïde de base
  2. Renseigner son degré
  3. Choisir si le simploïde devra être créé dans l'ensemble courant ou dans un nouvel ensemble
  4. Ajouter le simploïde grâce au bouton Add

Opérations de construction

Cette rubrique regroupe les différentes opérations de construction :

  • Cône
  • Produit cartésien
  • Identification
  • Dégénérescence

Opération de cône

Cette opération permet d'obtenir de nouveaux simploïdes à partir d'un point.
Pour appliquer cette opération il faut suivre les étapes suivantes :

  1. Créer un nouvel ensemble contenant seulement un point.
  2. Sélectionner ou créer l'ensemble simploïdal à partir duquel l'opération sera appliquée.
    Les simploïdes de cet ensemble ne doivent pas être de dimension supérieure à 2.
  3. Déplacer le point vers la position souhaitée.
    Le mécanisme de déplacement d'un simploïde est détaillé ici.
  4. Cliquer sur le bouton Cone pour finaliser l'opération.

Produit cartésien

Cette opération permet de réaliser le produit de deux ensembles simploïdaux.
Pour l'appliquer, sélectionnez deux ensembles puis cliquez sur le bouton Product.
Un message d'erreur s'affiche en cas d'échec.
Le mécanisme de sélection d'ensemble simploïdal est détaillé ici.

Opération d'identification

Cette opération permet d'identifier plusieurs simploïdes d'un même ensemble en un seul simploïde.
Pour l'appliquer, sélectionnez plusieurs faces de simploïdes de même type et de même degré puis appuyez sur le bouton Identify.
Un message d'erreur s'affiche en cas d'échec.
Le mécanisme de sélection de simploïde est détaillé ici.

Opération de dégénérescence

Cette opération permet de réaliser la dégénérescence d'un simploïde en un autre simploïde.

Il existe deux manières de réaliser cette opération :

  • Degenerate to point :
    Cette option permet de dégénérer n'importe quel simploïde en un point.
    Pour l'appliquer, sélectionnez un simploïde puis cliquez sur le bouton Degenerate to point.
  • Degenerate :
    Cette opération permet de dégénérer un simploïde en un de ses bords.
    Pour se faire, sélectionnez un simploïde et un de ses bords, puis cliquez sur le bouton Degenerate.
    Le mécanisme de sélection multiple de simploïdes et la reconnaissance des bords d'un simploïde sont détaillés ici.

Vue explosée

Cette option permet d'obtenir une vue explosée des simploïdes.
Pour choisir le facteur d'explosion, réglez le slider Exploded view factor.

Type de plongement

Le choix du type de plongement se fait à travers cette sous-rubrique.
Les différents types de plongement sont disponibles à travers le menu déroulant qui s'affiche après avoir cliqué sur le bouton Embedding.
En mode plongement de bézier, il est possible de régler la précision des surfaces et des courbes grâce au slider Bezier samples.

Menus

Cette partie de l'interface permet d'accéder à diverses fonctionnalités du logiciel.
Elle contient les sous-menus suivants : File, View et About.

File

Ce sous-menu regroupe les fonctionnalités suivantes :

  • New : Créer un nouvel espace de travail.
  • Save : Enregistrer le travail en cours (raccourcis clavier : Ctrl+S).
  • Save As : Enregistrer le travail en cours en donnant un nom au fichier. Ce dernier sera enregistré dans le répertoire spéficié sous le format choisi.
  • Open : Ouvrir un fichier déjà enregistré ou importer un modèle externe (raccourcis clavier : Ctrl+O).
  • Insert : Insérer un travail enregistré dans l'espace de travail actuel.

View

Ce sous-menu regroupe les fonctionnalités suivantes :

  • Orthographic view : Cette option permet de passer en vue orthographique.
  • Wireframe view : Cette option permet de passer en mode fil de fer. On y voit les triangles ou les carrés utilisés pour dessiner les surfaces.
  • Hide/Show control points : Option pour afficher ou cacher les points de contrôles.
  • View simploids for identification : Cette option permet d'afficher ou non les simploïdes compatibles pour l'opération d'identification lorsqu'on sélectionne un simploïde.
  • Render mode : Dans ce mode, seul les ségements et les surfaces sont affichés.
  • Hide/show toolbox : Cette option permet d'afficher ou de cacher la boîte à outil.
  • Hide/show content manager : Cette option permet d'afficher ou de cacher l'arbre des simploïdes.

About

Ce sous-menu permet d'accéder à ce manuel d'utilisateur ainsi que d'obtenir des informations sur le logiciel.

Arbre des simploïdes

La partie gauche de l'interface contient la liste des ensembles simploïdaux et des simploïdes.

Liste des ensembles simploïdaux

Un ensemble simploïdal contient la liste des simploïdes groupés par dimension.

La sélection d'un ensemble se fait par un clic-gauche sur l'ensemble souhaité.
La sélection multiple d'ensemble peut se faire de deux façons :

  • Maintenir la touche Ctrl puis clic-gauche sur les ensembles souhaités.
  • Maintenir le clic-gauche sur un ensemble et en déplacer la souris vers les ensembles souhaités.

Un clic-droit sur un ou plusieurs ensembles, affiche un menu contenant les fonctionnalités suivantes :
  • Rename : Permet de donner un nouveau nom à la sélection d'ensemble (raccourci clavier : F2)
  • Delete : Permet de supprimer la sélection d'ensemble (raccourci clavier Suppr)
  • Cut : Permet de couper la sélection d'ensemble (raccourci clavier Ctrl+X)
  • Copy : Permet de copier la sélection d'ensemble (raccourci clavier Ctrl+C)
  • Paste : Permet de coller un ou plusiers ensemble copiés ou coupés (raccourci clavier Ctrl+V)

La sélection d'un ou plusieurs ensembles met en évidence dans le menu central du logiciel les simploïdes y appartenant.

La fonctionnalité visible permet d'afficher ou pas les simploïdes d'un ensemble dans le canvas.

Liste des simploïdes

Pour voir les simploïdes d'un ensemble simploïdal, double-cliquez sur son nom ou cliquez sur le symbole +.
En déroulant par dimension on peut accéder aux simploïdes.

  • Dimension 0 contient les simploïdes de dimension 0 (les points).
  • Dimension 1 contient les simploïdes de dimension 1 (les segments).
  • Dimension 2 contient les simploïdes de dimension 2 (les faces).
  • Dimension 3 contient les simploïdes de dimension 3 (les volumes).
Le type de chaque simploïde est précisé derrière son nom.
Le symbole - désigne les bords du simploïde sélectionné.
Le symbole * désigne les simploïdes principaux de l'ensemble.
La lettre D désigne les simploïdes dégénérés.
La lettre d désigne les simploïdes vers lesquels celui sélectionné a été dégénéré.
Un clic-droit sur un simploïde permet d'accéder aux fonctionnalités Rename, Delete, Cut, Copy et Paste.

Le mécanisme de sélection multiple de simploïdes est le même que celui pour les ensembles simploïdaux.

Canvas

Le canvas est la partie se trouvant au centre de la fenêtre du logiciel.
Le rendu des simploïdes y est effectué et il est possible de manipuler la caméra.

Camera

La caméra permet la visualisation des simploïdes.
La caméra peut être :

  • Tournée en maintenant le clic-gauche sur le canvas tout en déplacant la souris.
  • Translatée en maintenant la touche Ctrl et le clic-gauche sur le canvas tout en déplacant la souris.
  • Rapprochée ou éloignée en tournant la molette de la souris.

Manipulation des simploïdes

La sélection d'une face, d'un segment ou d'un sommet d'un simploïde peut se faire à travers un clic-gauche sur ce dernier.
La sélection d'un simploïde principal se fait par la combinaison Alt + clic-gauche sur un simploïde.
Il est possible de sélectionner plusieurs simploïdes en maintenant la touche Ctrl.

Après avoir sélectionné un simploïde à travers le canvas ou l'arbre des simploïdes, les opérations suivantes peuvent être appliquées sur le simploïde :

  • Rotation : Clic-droit sur le simploïde et déplacer la souris.
  • Translation : Clic-droit sur le simploïde, maintenir la touche Ctrl et déplacer la souris.
  • Mise à l'échelle : Ctrl + tourner la molette de la souris.

Manipulation des points de contrôle

Un point de contrôle peut être déplacé en maintenant le clic-gauche sur ce point et en déplaçant la souris.
